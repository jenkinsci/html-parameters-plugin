Каждый атрибут id элемента должен начинаться с html-parameters-. Каждый токен CSS-класса должен начинаться с html-parameters- или jenkins-, в соответствии с разделом Stylesheets Jenkins Design Library (классы jenkins- допустимы в плагинах). Не используйте классы app- — они считаются внутренними для Jenkins и могут меняться без предупреждения.

Для цветов с учётом активной темы используйте классы палитры и CSS-переменные Jenkins из Design Library — Colors, а не жёстко заданные цвета.

Для оформления используйте поле Custom CSS, а не встроенные атрибуты style в этом шаблоне.

Чтобы выпадающий список <select> выглядел как в Jenkins, повторите разметку f:select: <div class="jenkins-select"><select class="jenkins-select__input" id="html-parameters-…">…</select></div> (атрибут id оставьте на select для сопоставления). См. Design Library — Select.
