Every element id must start with html-parameters-. Each CSS class token must start with html-parameters- or jenkins-, as described in the Jenkins Design Library Stylesheets pattern (plugin-safe jenkins- classes). Do not use app- classes; they are internal to Jenkins and may change without notice.

For colors that follow the active theme, use Jenkins palette classes and CSS variables from Design Library — Colors instead of hard-coded colors.

Use Custom CSS for styling instead of inline style attributes on this template.

For a themed <select>, use the same structure as Jenkins f:select: <div class="jenkins-select"><select class="jenkins-select__input" id="html-parameters-…">…</select></div> (keep id on the select for mappings). See Design Library — Select.
