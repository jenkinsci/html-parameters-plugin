Selectors must only target your parameter markup: every comma-separated selector must contain the substring html-parameters- (for example .html-parameters-panel or .html-parameters-root .jenkins-button). The rendered control is wrapped in a root element with class html-parameters-root.

Only @media at-rules are allowed; other @ rules (such as @import) are not permitted.

For theme-aware colors (light/dark and other themes), prefer Jenkins CSS variables and palette classes from the Design Library — Colors page instead of hard-coded color values.
