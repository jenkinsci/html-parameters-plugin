Селекторы должны затрагивать только разметку параметра: каждый селектор в списке через запятую должен содержать подстроку html-parameters- (например .html-parameters-panel или .html-parameters-root .jenkins-button). Разметка выводится внутри корневого элемента с классом html-parameters-root.

Разрешено использовать только at-правила @media; остальные @-правила (например @import) запрещены.

Для цветов с учётом темы (светлая/тёмная и др.) используйте CSS-переменные и классы палитры Jenkins из раздела Design Library — Colors, а не жёстко заданные значения цвета.
